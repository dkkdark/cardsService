# cardsService

Бэкенд для приложения: https://github.com/dkkdark/CardsServicesApp

Использована библиотека gorm. Для создания базы данных использовался PostgreSQL. 

Каждый метод в handlers обращается к методам из publlic. Они выполняют select-запросы, update, insert, обращаются к функциям и процедурам БД.
